## 核心能力矩阵

| 维度 | 当前实现 | 扩展路线 |
|:-----|:--------:|:--------:|

| **安全审计** | 🔜 Middleware | ✅ 完整链路追踪 |

## 设计哲学

> **"Progressive Intelligence, Zero Compromise on Security"**

| 原则 | 描述 |
|------|------|
| **分层解耦** | Agent / Skill / Tool 三层分离，各司其职 |
| **可插拔架构** | Middleware、Backend、Skill 即装即用 |
| **零信任安全** | 每次操作独立鉴权，可读范围最小化 |
| **沙箱隔离** | 自动化任务在隔离环境执行，不影响主流程 |
| **可观测性** | 完整工具调用审计日志 |

## 高并发 & 水平扩展

```
                         ┌─────────────────┐
                         │   Load Balancer │
                         │  (K8s / HAProxy) │
                         └────────┬────────┘
                                  │ horizontal scaling
           ┌─────────────────────┼─────────────────────┐
           ▼                     ▼                     ▼
    ┌──────────────┐      ┌──────────────┐      ┌──────────────┐
    │   Worker 1   │      │   Worker 2   │      │   Worker N   │
    │ (uvicorn)    │      │ (uvicorn)    │      │ (uvicorn)    │
    └───────┬──────┘      └───────┬──────┘      └───────┬──────┘
            │                     │                     │
            ▼                     ▼                     ▼
    ┌─────────────────────────────────────────────────────────┐
    │                   StoreBackend (Redis/DB)               │
    │              namespace = (user_id, thread_id)            │
    └─────────────────────────────────────────────────────────┘
```

| 策略 | 实现方式 |
|------|----------|
| **无状态 Agent** | 核心逻辑无状态，状态通过 `context` 参数传入 |
| **多 Worker** | `uvicorn --workers N` 单机多进程 |
| **多实例部署** | 前置负载均衡 + 分布式 StoreBackend |
| **会话隔离** | 基于 `thread_id` 隔离，会话间无共享状态 |
| **限流重试** | LLM API 自动指数退避重试（默认 6 次） |

**核心原则**：Agent 无状态，可水平扩展；状态存储按需替换 Backend 实现。

## 技术栈

| 层级 | 技术 |
|------|------|
| Agent Runtime | ReAct 循环 + Context 管理 |
| Skills | Markdown 指令模板（按需加载） |
| Tools | 无状态原子函数 / CLI 可独立测试 |
| Chain | Hyperliquid SDK / CoinGecko API |
| API Layer | FastAPI + SSE 流式响应 |
| 扩展 | Middleware / Sandbox / Backend (规划中) |

## 快速开始

```bash
# 安装
uv sync

# 启动 Agent（单机）
uv run python -m src.agent.run

# 启动 API Server（支持多 Worker）
uv run uvicorn src.agent.fastapi_server:app --workers 4 --host 0.0.0.0 --port 8000
```
